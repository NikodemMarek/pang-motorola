Gra Pang na Motorola Science Cup 2022

Implementacja gry Pang w TypeScript, stworzona na konkurs Motorola Science Cup 2022.

Gra wykorzystuje silnik graficzny PixiJS. Interfejs graficzny jest zbudowany przy użyciu biblioteki React.

Uruchamianie

Sklonuj projekt i wejdź do folderu z projektem

  git clone https://github.com/NikodemMarek/pang-motorola.git
  cd pang-motorola

Pobierz zależności

  npm install

Uruchom serwer

  npm run start

lub uruchom aplikację przez NW.js

  npm run run-nw

Testy

Testy są uruchamiane automatycznie podczas buildu.
Dodatkowo mogą zostać uruchomione za pomocą komendy

  npm run test

 
Wykorzystanym frameworkiem testowym jest Jest.
Ponieważ do renderowania gry wykorzystane są zewnętrzne biblioteki, testowana jest tylko fizyka w grze.
